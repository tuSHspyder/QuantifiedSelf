QuantifiedSelf App Version 1.0 17/03/2022

Usage notes
----------------------------------
- This application tracks daily life tasks/goals with the help of data and shows the trend graph of the data.



How to run the app
----------------------------------
- Python must be installed in the machine in order to run the app.

- Some dependencies are required to run the app.py file(Read requirements.txt)
	- To install dependencies
		-Run command in terminal '$ pip install -r requirements.txt'

- After installing the dependencies, run the app
	- Open terminal then go to the project path.
	- Run command '$ python app.py'
